Projects

Flagship · Open source

Mise-en-Place

A multi-agent meal-planning system for athletes — built to show what production-grade agent orchestration actually looks like.

A LangGraph graph coordinates specialized agents to plan a full week of macro-targeted meals, then pauses for a human-in-the-loop review before anything is saved. Custom MCP servers expose the user's pantry and grocery state to the model; per-day generation runs in parallel (7-day plans dropped from ~70s to ~33s). Every run is traced in LangSmith. React + FastAPI front the whole thing, with measured-TDEE analytics derived from real weight logs.
